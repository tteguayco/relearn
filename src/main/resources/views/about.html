What is it?

Relearn is a software tool thought to help students when learning Relational Algebra, which is a relational query language proposed by Edgar F. Codd in his 1970 paper called "A Relational Model of Data for Large Shared Data Banks".

Understanding how Relational Algebra really works is important due to it has provided the underlying theoretical concepts of many commercial query languages for relational databases used nowadays, such as the well-known SQL. Because of this, a student who masters Relational Algebra is more likely to retrieve the correct information from relational databases and to write better queries, in terms of efficiency, in SQL.

Would you like to collaborate?

  • If you detect any error while using the app which should be fixed, you can open a new issue on GitHub to let the contributors know about it. Please, give as many details about the incidence as you can: screenshots, steps to reproduce the error, etc.
  • You may also want to make improvements on the app on your own or to contribute by adding new functionality. If so, you can clone the project's repo, make all the changes you want and then send them through a Pull Request to be reviewed by the contributors.
